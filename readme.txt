Instructions

1. Unzip the provided project folder.

2. Install Python Dependencies within the project directory
	pip install -r requirements.txt

3. Import the database HorizonTravels.sql

4. Update the database configuration in dbfunc.py if necessary.

5. Run the website by navigating to the 'project-folder' directory and run the Flask server using the following command.
	For Windows : python project-folder/app.py
	For Mac : python3 project-folder/app.py

6. The website should be running on 127.0.0.1:5000 by default.
